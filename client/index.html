This website gets a random song and chart from the game beatmania IIDX 25 CANNON BALLERS (among other things)

About

This website was created by Michael Berger to help beatmania IIDX 25 CANNON BALLERS players choose songs.

It introduces a web API that allows other developers to search and retrieve song information from the game.

The data was scrapped from the master list of IIDX 25 songs on RemyWiki.
