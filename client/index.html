DPS-KUN

A IIDX Companion Website

This website gets a random song and chart from the game beatmania IIDX 25 CANNON BALLERS

Included is a fully-functional API that lets users search a database of all currently-available songs.

Users can also create "Playlists" of songs that will be shared publicly on a separate web page.

Currently only getting a random song from the API and retrieving a song list are implemented.

To test getting a random song, append "/api/?random=1" to the address bar.

To test retrieving a song list, go to the address "/api/?list=X,Y,Z" where X, Y, and Z are numbers below 1208.
