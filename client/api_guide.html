How To Use The DPS-KUN API

Below are the requests that you can make and what they retrieve:

/api/?random

Gets a random song from the database and returns it as a JSON object.

/api/?search=[term]

Searches the database for songs containing the search term and returns a JSON object with matching results. Searches are ranked by song name first and song artist second, followed by the other song properties.

/api/?list=[ID#1,ID#2,ID#3,etc.]

Returns a JSON object containing an array of songs, which are defined by the given parameter of comma-separated song IDs. To find what songs have which ID, use the "Song Search" page.

/api/?playlists

Returns a JSON object containing an array of all the published DPS-KUN playlists. The playlist JSON object consists of properties "name" (literally the name of the playlist) and "songs" (an array of the songs in the playlist).

/api/addlist

A POST command to add a playlist to this website. Expects the posted message to be in this format:

name=[name]&list=[ID#1,ID#2,ID#3,etc.]

ERROR CHECKING

If an API call of either GET or POST fails, the returned parameter is a JSON object with an 'error' property. This property is a string that explains why the GET/POST could not happen.
